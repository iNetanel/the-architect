# Code of Conduct

The Architect follows the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.

The full text of the Code of Conduct is available at:

**https://www.contributor-covenant.org/version/2/1/code_of_conduct/**

## Summary

All participants in this project — contributors, maintainers, issue reporters,
and users — are expected to:

- Treat everyone with respect, empathy, and kindness.
- Welcome differing viewpoints and constructive feedback.
- Focus on what is best for the project and the community.
- Show patience with newcomers and people learning.

Harassment, personal attacks, discriminatory language, and other unprofessional
conduct are not tolerated in any project space (issues, pull requests,
discussions, commit messages, or private communication related to the project).

## Scope

This Code of Conduct applies within all project spaces and when an individual
is officially representing the project in public.

## Reporting

If you witness or experience unacceptable behaviour, report it privately to:

**[inetanel@me.com](mailto:inetanel@me.com)**

Reports are handled confidentially. You will receive acknowledgement within
72 hours and a decision or follow-up within 7 days.

## Enforcement

The project maintainer is responsible for clarifying and enforcing this Code
of Conduct and will take appropriate action in response to any behaviour
deemed inappropriate, threatening, offensive, or harmful. Consequences may
include a warning, a temporary ban, or a permanent ban from the project.
